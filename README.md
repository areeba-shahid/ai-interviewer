# 🎯 AI Interviewer - Practice Interviews with AI

![AI Interviewer Banner](https://via.placeholder.com/1200x300/1e40af/ffffff?text=AI+Interviewer)

A full-stack MERN application that helps candidates practice for technical interviews using AI-powered questions, voice responses, and real-time feedback.

## 🚀 Live Demo

- **Frontend**: [https://ai-interviewer.vercel.app](https://ai-interviewer.vercel.app)
- **Backend API**: [https://ai-interviewer-api-bbmm.onrender.com](https://ai-interviewer-api-bbmm.onrender.com)

## ✨ Features

### 🎤 Voice-Based Interviews
- AI speaks questions using Web Speech API
- Users respond with voice, automatically transcribed
- Natural conversation flow with AI

### 🤖 AI-Powered Questions
- Dynamic questions generated by Google Gemini AI
- Personalized based on role, experience, and tech stack
- Real-time answer evaluation with scoring

### 🔒 Anti-Cheating Measures
- 📹 Camera monitoring during interview
- 🖥️ Fullscreen enforcement
- 🚫 Tab switch detection (terminates interview)

### 📊 Comprehensive Dashboard
- Track interview history and scores
- View strengths and areas for improvement
- Performance charts and statistics

### 👤 User Management
- Google Authentication via Firebase
- 6-step profile setup wizard
- Customizable settings and preferences

### 📱 Fully Responsive
- Works on desktop, tablet, and mobile
- Beautiful UI with Tailwind CSS

## 🛠️ Tech Stack

### Frontend
- **React 18** + Vite
- **Tailwind CSS** for styling
- **Firebase Authentication** (Google Sign-in)
- **Recharts** for data visualization
- **React Router v6** for navigation
- **Axios** for API calls

### Backend
- **Node.js** + Express
- **MongoDB Atlas** with Mongoose
- **Google Gemini AI** for question generation
- **Firebase Admin** for auth verification
- **JWT** for secure routes

### Deployment
- **Frontend**: Vercel
- **Backend**: Render
- **Database**: MongoDB Atlas

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas account
- Firebase project
- Google Gemini API key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/areeba-shahid/ai-interviewer.git
   cd ai-interviewer
   Install backend dependencies

bash

npm install

bash

npm install
Set up environment variables

Backend (server/.env):

env

MONGODB_URI=your_mongodb_connection_string
